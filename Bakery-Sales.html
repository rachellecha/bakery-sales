As we can see It seems that people really like to buy Traditional Baguettes during the summer months! People seem to like Baguettes! This information is important because then the baker can know to prepare more baguettes in the summer. I wonder what the average number of baguettes a person buys in a single trip is? Since Baguettes are big, I bet the average person only buys 1 at a time.

Hypothesis Testing

null hypothesis: people buy one baguette (x = 1)

alternative hypothesis: people dont buy one baguette (x != 1)

alpha = 0.05

mu <- mean(bakery_baguette$Quantity)

x <- 1

n <- length(bakery_baguette$Quantity)

std <- sd(bakery_baguette$Quantity)

z <- (x-mu)/(std/sqrt(n))

print(z)
## [1] -159.8403
p_val <- pnorm(z)*2

print(p_val)
## [1] 0

Analysis

The p-value is 1. This means that we reject the null hypothesis in favor of the alternative hypothesis. It looks like people dont buy just one baguette at a time. This is useful for the baker to know so he can make sure to bake a lot of traditional baguettes!
